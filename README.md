# 可视化组件教程样例

### 模板样例简介

本模板为imoka可视化组件教程样例

### 如何运行

* 安装命令行工具

```
# npm
npm install -g @imoka/imoka-cli 

# yarn
yarn global add @imoka/imoka-cli
```

* 执行操作

```
git clone git@github.com:akoomitech/example_course.git

cd example_course

npm install

npm run serve
```

模板启动后， 打开浏览器访问 https://localhost:8082
